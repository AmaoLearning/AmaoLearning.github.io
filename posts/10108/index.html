马克思 第一卷第一版序

在科学上面是没有平安的大路可走的，只有那在攀登上不畏劳苦不畏险阻的人，有希望攀到光辉的顶点

Das Kapital: Kritik der politischen Ökonomie，中文译名《资本论：政治经济学批判》是我自高中接触简单的哲学以来就一直想读的书。它全书分为三卷，第一卷主要讨论马克思认为的经济学的原子——商品和他的剩余价值学说，在第一版序中他自述第二卷讨论资本的流通过程和总过程的各种形态，第三卷讨论学说史。本次我希望借长假空闲之余读完第一卷，其次为使过程不至于太单调，过程中我会以时间顺序简单记录书中一些重要的观点和我个人的感想。若博客中我的理解存在与原书不相当之处，欢迎评论指出。

采用的中文译版来自郭大力、王亚南译，2011年三联书店编。

25.06.16 下午 梅雨见晴

恩格斯 英译本第一卷编者序

任一种科学，每当有新解释提出时，总不免要在这个科学的术语上，发生革命。

恩格斯在编者序中解释了书中有些名词跟日常生活、普通经济学上的用法不同。马克思引用别人的话时并非支持他的观点或用作自己观点的证明，而是说明这种主张最初是何时何地何人提出的，是一种科学史的注释。

第一章 商品

商品的二因素：使用价值和价值

  • 使用价值：来自物体的效用，因而具有不可与物体分离、与人的劳动量无关的性质；是交换价值的物质承担者。
  • 交换价值：表现商品间的交换比例，是时刻变化的，又是可传递的。
  • 价值：不考虑商品的使用价值后，抽象的人类劳动是价值的唯一衡量。它是商品交换价值上的共通物，交换价值是价值必然的表现方式。
  • 同一的人类劳动力：排除个人生产是否熟练的因素，一个人的劳动力生产一个商品所必要的劳动时间比平均所必要的劳动时间少或持平，他就是同一的人类劳动力。
  • 社会必要劳动时间：社会标准生产条件下，用社会平均的劳动熟练程度与强度，生产一个使用价值所必要的劳动时间。它决定了使用价值的价值量。而必要劳动时间又随社会劳动生产力变化，因而价值随劳动生产力反向变化。
  • 特殊情况：无价值的有使用价值物品如空气、自然草地等，为自身目的使用自身劳动生产或者不作为商品流通的有使用价值物品如自己种自己吃的菜、中世纪农民上供的粮食等，需要劳动但没有使用价值的物品，种种不属于商品。

25.06.18 中午 晴暑

在商品中表现的劳动的二重性

  • 有用劳动：使生产物成为一种使用价值的劳动。生产物的使用价值性质不同、有用劳动性质不同时才能当作商品相互交换，比如上衣不与上衣交换。它还必须是一种有目的的生产活动，使自然物质适合于某种人类欲望。
  • 社会分工：不同种的有用劳动，一个工厂内的分工不算，必须涉及个人生产物间的交换。
  • 单纯的平均劳动：在一定的社会内（可能国与国不同）是确定的，可以作为劳动换算的尺度，复杂劳动可以看作倍加的单纯劳动。因为生产活动剥离劳动的有用性质后都是人类劳动力的支出，资本主义社会中根据变化的劳动需要，一部分劳动的形态会发生变化，也就是人类劳动力的支出形态变化。
  • 商品虽然使用价值和有用劳动性质不同，但仅考察价值时应看作等质的劳动凝结物，都具有人类劳动。因此定量衡量商品价值时，依据的是劳动力的支出时间。
  • 劳动的二重性体现在它表现为价值的单纯劳动，和创造使用价值的有用劳动。这二者是可以相互对立运动的。生产力指有用的具体劳动的生产力，它增加时有用劳动也增加，却不会影响单纯劳动，这是因为，同一劳动在同一时间内所提供的价值量不变，但使用价值量会正比于生产力。但是生产力增大会增加相同劳动时间生产的使用价值量，换言之，减短相同使用价值生产所必要的劳动时间，从而会减少等量使用价值的价值。

价值形态或交换价值

  1. 单纯情况：

    • 商品有自然形态和价值形态的二重形态。与使用价值杂多的自然形态相反，商品有共同的价值形态——货币形态。
    • xA=yB中，A是价值被表现的商品，处于相对价值形态；B是价值赖以表现的商品，处于等价形态。A之所以能与B等同，是因为他们有着共通的抽象人类劳动的内在性质。
    • 相对价值随生产力变化而变化，如生产麻布的织物劳动生产力下降到一半，而生产上衣的裁缝劳动生产力不变时，前者相对后者的价值应当加倍。同样，可能前者生产力不变，而后者变化；也可能前后者向同方向同比例、不同比例变化。

    • 等价形态放在某商品如上衣上时，商品自身的价值量被抹消了。商品不能为自身的等价。而上衣之所以能作为衡量麻布的等价形态，它必须能跟麻布直接交换。等价形态还有本质等一（不然就不能交换，这里的等一物就是平等的人类劳动）和私人劳动两种特征。

      马克思评价说奴隶制社会的亚里士多德虽然察觉到了商品的平等关系，但碍于商品支配关系和劳动力的不平等，没能发现平等关系中的等一物——人类劳动。

    • 商品的价值必须放在跟其他商品发生价值关系的情况下考察，是不能孤立存在的。

    • 在A=B的价值关系中，A的自然形态扮演使用价值，B的自然形态扮演价值形态。因此在二商品的关系上表示出商品的二重性。

  2. 总体的或扩大的情况：

    • 扩大的相对价值形态：某件商品相对价值可以表现在整个商品世界的任一商品上，其交换关系由商品的价值量决定。
    • 特殊等价关系：在交换关系中充当等价形态的商品，其自然形态变得特殊，其内含的具体有用劳动则是人类劳动的一种特殊的实现形态。
    • 缺点或局限性：
      • 商品扩大的相对价值形态是无穷无尽的锁链环，锁链之间相互交错；每个新商品又会形成新环并延长每个环；商品间的相对价值形态不同，没有共通。
      • 每个特殊的等价形态相互排斥且局限；它们的总和才是人类劳动整体的实现形态，但是没有统一。
  3. 一般的价值形态：
    • 一般等价形态：
      • 用一种实际与其他商品分开的商品来表现商品世界全体的价值。它具有与一切商品等同的共通性、所有新旧商品服从的社会性，还为其他商品提供了价值的可比较性。
      • 此外，生产一般等价商品的具体有用劳动也成为了人类劳动一般化的实现形态，将劳动的具体形态和有用性质抽象成了所谓的人类劳动力的支出，其他具体劳动经过此种还原后能够与它等同。
      • 反过来，一般等价商品只能以扩大的相对价值形态表现自己的相对价值。
    • 相对价值形态的发展表现为等价形态的发展：从单纯的相对价值形态、扩大的相对价值形态，再到一般的相对价值形态，价值关系从一表现一，到多表现一，最后到一表现多。
  4. 货币形态：
    • 金成为社会公认的一般等价商品。
    • 价格形态：相对金的单纯相对价值形态。

综上，单纯的商品价值形态是货币形态的胚芽。

商品拜物教性质及其秘密

  1. 商品的神秘性：人们把人类劳动的社会性质看成了劳动生产物自身的对象性质，或者它的社会的自然性质。实际上商品间的机制关系只是人与人之间的社会关系，生产者因为互相交换劳动生产物才有社会接触。商品拜物教就是将这种社会关系全然看成是物与物之间的关系，认为商品是独立的存在，从而商品交换时的价值似乎不受交换者控制，商品价值看似是偶然的。但实际上是由劳动时间规定价值量的。
  2. 私人劳动之于社会：
    1. 有用性：劳动生产物不仅有用，对于他人也有用。
    2. 平等性：不同的生产物有共通的价值性质的形态。
  3. 货币主义者倾向于将金银等货币看成是具有特别社会性质的自然物，而非社会生产关系的代表，这是一种拜物教思维。

第二章 交换过程

  1. 使用价值与价值孰先孰后是各有道理的：
    1. 因为商品对生产者没有使用价值，对他人有，所以商品在能够实现为使用价值之前，必须通过交换实现为价值。
    2. 而商品在生产完成前，它又必须被投入对他人有用的劳动，即实现使用价值。但生产物对他人是否有用又必须通过交换来证明。
  2. 货币是交换过程的必然的结果。交换的扩大使得商品的使用价值与价值的对立逐渐明显，将这种对立在外部表现出来的必要，就要求了特殊商品作为公认的等价形态。而商品变多和交换的扩大也使这种特殊商品的公认程度越来越大，因而成为货币。
  3. 使用对象成为交换价值首先要求它对所有者非使用价值，其次所有者是独立的物品私有者。前者保证物品可以让渡，后者排除了共产体内部=
  4. 货币作为价值量的现象形态，要求是各地皆有一致性质、且其价值量的差别只与量有关，金银天生符合这些要求。

第三章 货币或商品流通

25.06.22 夜 大雨

价值尺度

  1. 价值尺度：货币是价值尺度，而商品内在的价值尺度是劳动时间，货币是其现象形态。
  2. 以金表现商品价值的行为是观念的，货币用作价值尺度时它只是想象的。
  3. 货币作为价格标准时，它是要以某定量的金作为尺度单位，计量金量本身。
  4. 货币名称最初是某种贵金属的重量名称（计量金或其他贵金属量），然而逐渐与重量分离：
    1. 外国货币名称必然不能与国内重量名称相同；
    2. 财富发展使得较贵重的金属取代较不贵重的金属作为价值尺度，而名称不变。
    3. 帝王的货币伪造政策曾改变铸币的原重量，仅名同而质不同。
  5. 金的价值变动不会妨碍金作为价格标准，因为金量之间比例不变。
  6. 用单纯的相对价值表现解释商品价格变动：
    1. 商品价格提高：
      1. 商品价值提高，货币价值不变；
      2. 货币价值跌落，商品价值不变；
    2. 商品价值跌落：
      1. 商品价值跌落，货币价值不变；
      2. 货币价值提高，商品价值不变；
  7. 商品价格与货币的交换比例，不一定是商品价值量的指数：
    1. 商品价值量表现的是其生产过程中与社会劳动时间的关系；
    2. 价格可以与价值量相背离，比如良心名誉、未耕地没有人类劳动也可以卖出价钱等。这可以用相对价值形态解释，即价格表现的是一定量的货币（如一盎司金）能直接与商品交换，而非商品能直接与货币交换，商品必须先转化成实际的货币。

流通手段

  1. 商品的形态变化：

    1. 应当把交换过程，即使商品从非使用价值的人手里转移到有使用价值的人手里，看作社会的一种代谢机能。
    2. 商品的交换过程有如下形态变化：商品——货币——商品
      1. 商品——货币，卖的过程，商品的开始形态变化。社会分工使得欲望有多方面，而劳动常常是不能直接满足欲望的另一方面。
      2. 商品价格或其观念的价值形态实现的同时，货币的观念的使用价值也实现了。即商品与货币同时互相转化，买卖在商品所有者和货币所有者双方是同时进行的。
      3. 在商品未让渡时，金是商品的价值形态、是观念的货币或价值尺度；当它被让渡后，金就变成了现实的价值形态，而商品的使用价值就丧失了，取而代之的是商品的交换价值。
      4. 货币——商品，买的过程，又称商品的终了形态变化。货币是绝对可以让渡的商品，商品先换成货币，货币又可以分开来买许多商品，故它的终了形态变化是其他许多种商品的开始形态变化之总和。
    3. 以上描述的种种买卖循环和商品形态变化构成了商品流通，在流通中，许多商品进场许多退场，退场时空缺的位置由货币补上了，流通使货币不停地奔走，第一个卖家卖时失去的商品又会被生产的新商品补充。
    4. 货币流通时卖与买的同一性：商品不卖则不能被买，商品就无用；没有人买就没人能卖，卖出商品的人也不必要马上买。这与直接商品交换中卖即买有本质不同。
    5. 货币作为商品流通的媒介，成为了流通手段。但其流通手段的机能仅因为它是独立化的商品价值或者商品的形态运动。

  2. 货币的通流

    1. 商品流通过程中货币的运动是从一个商品所有者到另一个，是越来越远的。
    2. 货币的通流指同一过程的不绝的单调的反复，只有从买者手里到卖者手里这一个过程。
    3. 对某个商品而言，开始形态变化包含商品与货币两者的运动，终止形态变化却只表现为货币或其价值形态的运动了，因为该商品自身已经到了第一个买家的手里，对卖家来说只有货币了。因此尽管商品流通由两个相反的过程，但货币的运动是永远连续、同一的。
    4. 问题：流通界究竟有多少货币/流通手段？
      1. 商品价格的总额规定了商品流通过程中必要的流通手段量。
      2. 贵金属的不绝流入必然使货币的价值尺度的价值跌落，使价格变化。但这种变化并不完全是因为货币流通量增加而导致的。
      3. 假设：商品价格不变，该假设在一瞬间内的分析是合理的，则商品价格总额由流通的商品量决定。反过来商品量一定时，价格总额由价格决定，又决定流通手段量。
      4. 因为同一货币可以时序地作为许多商品的流通担当者，我们需要考虑货币通流速度。
      5. 关键法则：流通的货币量=商品价格总额 / 同名称货币的通流次数（一定时间内，或通流效率）
      6. 假定商品价格总额不变，则流通的货币量与货币流通次数反相关。某枚通流变快的货币能使其他货币通流变慢或者退出流通界。
      7. 流通中变化的三个关键要素，变动的价格、流通商品量和货币通流速度，它们与价格总额和流通手段量的关系如下：
        1. 商品价格不变，流通手段量因流通商品量增加或货币通流速度减小，而增加；反之减少。
        2. 商品价格增加，流通商品量减少或者货币通流速度增加，流通手段量可以不变；反之可以减少或增加。
      8. 各种要素的变动相互抵消使得流通货币量的变化并不快，激烈变化如经济危机的情况是周期原因。

  3. 铸币：价值记号

    1. 金铸币在通流中，会渐渐磨损，因此同名称金铸币实际重量不同，故实际价值不等。充当流通手段的金与充当价格标准的金背离了，这种分离暗示了用其他材料的记号来代替金货币的可能。
    2. 银和铜曾被引入代替极小重量的金，它们急速流通，极易磨损，其价值仅由国家法律规定。
    3. 纸币是铸币机能与其价值脱离的完全外显。纸币的发行额被无纸币时必要的金银流通量所限制，就算是超过了，也只能代表不变的必要金量。
    4. 与商品价值的关系：商品价值观念地由金量表现，金量象征地由纸币表现。
    5. 货币必须有客观的社会的妥当性，纸币则是由国家强制力背书的，所以只能在本国限界或其流通领域内发生效力。

货币

  1. 贮藏
    1. 商品卖者变为货币贮藏者。不卖而能买，因为以前有卖而不买之时。
    2. 货币因为可以直接和任何商品相交换，这种质的无限制导致了货币贮藏冲动的无限制。
    3. 金银商品的贮藏一方面是社会激荡时期金银的供给来源，另一方面又是流通界货币流入和溢出的水道。
  2. 支付手段
    1. 债务来自先购买后支付，买者仅以货币或未来货币的代表者的资格来购买。售卖者成了债权者，购买者成了债务者。货币也就成了支付债务的手段，简称支付手段。
    2. 现在商品流通过程中商品与货币的价值不是同时相等的，即商品已实现了使用价值，但其价值还未全部实现，或者说，在支付日期之前，商品形态变化停在了开始形态变化的结束。
    3. 实现引起所有债务的商品的价格总额所必要的货币量依赖于支付手段通流速度，它受制于
      1. 债权与债务关系的连锁。
      2. 各支付日期间的间隔。
    4. 必要的支付手段量有特殊的清算机制，即相互对照债务后只需要支付债务的余额，这大大减少了支付手段的必要量。
    5. 金融恐慌是某机构因某种原因发生扰乱时，货币必须立即地、突然地从观念的计算量变为现款，这是来自社会支付的连锁和清算组织的发展。
    6. 一定时期内通流货币总额=（待实现的商品价格总额 + 到期的支付总额 - 互相抵消的支付总额）/ 同名货币作为流通手段和支付手段的总通流次数。
    7. 信用货币出自货币充当支付手段的机能，现实表现为流通的债务证券，即债权的转移。
    8. 每个国家都有总支付的日期，它必须根据生产条件如季节变化等来规定，赋税、租金都是同样的道理。此时大量的货币会集中结付，因而引起一种周期的表面的扰乱。

世界货币

25.06.25 夜 晴

配第 爱尔兰的政治解剖

货币不过是国家身体的脂肪，过少，使身体生病，过多也障碍身体的运动。

配第 货币问答

铸币过多时，怎么办呢？我们可以把最重的铸币镕解，把它化作金银什器食器和用器；或把它输出到需要这些东西的地方；或把它输出到利息率很高的地方去生利息。

世界市场上只有金银贵金属作为价值尺度，即世界货币。世界货币充作一般支付手段时，重要功能是结算国际贸易差额。国家不仅为国内流通和支付贮藏一个准备基金，同时也要为世界市场的流通贮藏准备基金。商品生产流通发达的国家，银行准备库中贮藏的货币应该大为超过其机能所必要的最低下限。如果不超过机能平均水准，可能说明商品流通迟滞。

第二篇

第四章 货币的资本化

资本的总公式

25.06.26 夜 晴

  1. 商品流通的最后产物是货币，货币又是资本最初的现象形态。
  2. G——W——G（货币——商品——货币）形态变化运动是为卖而买
    • 这种方法流通的货币在性质上是资本，只是垫支，同一件商品会使货币回到原出发点来。除非买得的商品卖不出去，不然这种运动是无休止的。
    • 以买为始，以卖为终，刚好与简单的商品流通顺序相反。
    • W——G——W的最后目的是使用价值，货币的支出无关于其回来，其满足了消费欲望；G——W——G的动机仅为交换价值，并且目的仅为量而非质的变化，其欲望为货币自身的增殖。
    • G——W——G中卖出得到的价值超过原价值的追加量——剩余价值。
    • 资本家是G——W——G运动的有意识的担当者，其动机为抽象财富之递增的占有，使用价值绝非其直接目的，而是不息的牟利。他与货币贮藏者的区别是反复将货币投入流通来追求价值。
    • G——W——G运动中价值成了过程的主题，商品与货币二者都只是价值的不同存在方法（依我个人理解，这是因为它们对资本家都没有直接的使用价值）。价值产生价值。价值自身在不断的形态变化、不断的增殖中的同一形态是货币形态。
    • 上述为卖而买的过程表现的是资本的特殊形态——商人资本，资本还有货币化为商品再卖得货币得产业资本，和缩减了商品步骤的生息资本。
  3. 我们把G——W——G’当作资本的总公式。

资本总公式的矛盾

资本或者说剩余价值的出现，不在于单纯的商品流通中卖得贵或买得便宜，因为单纯的商品流通是等价物的交换，卖得贵获得的额外价值必然因为买得贵而损失，买得便宜同理。但是剩余价值只能依存于商品流通，因为没有流通也就没有卖和买，也就不能完成价值增殖的运动。（流通是商品所有者相互关系的总和，流通之外商品所有者仅和他自己的商品有关系，在流通之外提高商品价值的方法必然是商品所有者在其上附加了新的劳动/价值，但是离了流通他无法跟其他商品所有者接触，也就不能使价值增殖。）在流通间一定存在一系列狡诈的中间环节。

劳动力的买和卖

25.06.27 下午 雷阵雨

  1. 劳动力是一种特殊的商品，它的使用价值能够创造价值。
  2. 劳动力在市场上表现为商品有两个必要条件：
    • 劳动力所有者必须能够自由地出卖劳动力。买卖双方是平等的商品所有者。劳动力所有者应该只出卖一段时期的劳动力，因为一次卖完相当于成为奴隶，反而从商品所有者变为商品。
    • 劳动力所有者不能出卖本人劳动生产的商品。商品生产者除了劳动力之外还需要生产手段和生活资料，劳动力出卖者是只有劳动力的人。他们的存在是出于历史原因（？），是社会分工的发展程度足以使使用价值和交换价值分裂（？），马克思认为暂且不用讨论这些。
  3. 劳动力的价值同样决定于生产它和再生产它所必要的劳动时间，具体来说是维持劳动力所有者和补充者（儿女）所必要的生活资料的价值，有时还需要附加上教育或训练费。这种价值的多少主要依存于该国家自由劳动者阶级的习惯和生活要求，有着历史的和道德的要素。
  4. 简单计算生产劳动力每日平均所需的收入=（365 x 每日需要的商品量 + 52 x 每周需要一次的商品量 + 4 x 每季需要一次的商品量 + …）/ 365，每日平均所需收入的价值是生产劳动力所需要的支出，同时也是劳动力价值的最低限界。
  5. 劳动力的使用价值的让渡与它实际上的运用在时间上是分离的，劳动力购买者的货币也通常是作为支付手段以一定周期付给劳动者的，即劳动者把劳动力的使用价值垫支给了资本家。这直接体现为，资本家破产时，劳动者应领而未领的工资就直接损失了。
  6. 就目前我们讨论的范围内，劳动力买卖双方都是自由、平等、有所有权和自私的。

第三篇 绝对剩余价值的生产

第五章 劳动过程与价值增殖过程

劳动过程

25.06.29 夜 晴

  1. 劳动：劳动力的使用.
  2. 劳动过程的基本要素：
    1. 人类的目的活动（劳动）
    2. 劳动对象：原料是曾经过去的劳动滤过的劳动对象，一切原料都是劳动对象，但不是一切劳动对象都是原料。
    3. 劳动手段：劳动者用在将劳动传导到劳动对象上的传导物。劳动手段的使用与创造，其萌芽见于一些动物，但特为人类劳动过程的特征。它还是划分经济时期、人类劳动力发展程度、社会关系的指示物。广义上说应该包含进行劳动过程所必要的一切对象条件。
  3. 劳动过程是由人的活动，用劳动手段，在劳动对象上引起预先企图的变化。生产观点上，劳动手段与劳动对象表现为生产手段，劳动表现为生产过程。
  4. 过去劳动的生产物，可以当作生产手段，可以作为多种其他劳动过程的原料，可以当作补助材料（燃料、煤炭等），可以兼做劳动手段和原料（如家畜，加工与肥料），可以兼做消费品和原料（如葡萄，可称为中成品）。
  5. 生产消费：生产物当作劳动的生活资料被消耗。生产物是劳动过程的结果又是劳动过程的存在条件。不参加劳动过程的机械无用甚至会腐朽，在劳动过程中被消费可以当作新生产物的构成要素。
  6. 劳动过程与人类生活形态无关、与劳动者之间的关系无关，无论在奴隶主，还是资本家手中工作，都是共通的劳动过程。因此，被雇佣的劳动者虽为资本家利益工作，劳动过程的一般性质不会改变。
  7. 劳动过程被资本家消费劳动力的过程时有两点现象：
    1. 劳动者的劳动在资本家的管理下，合目的且节省。
    2. 生产物是资本家的所有物。

价值增殖过程

  1. 商品是使用价值与价值的统一，商品生产过程是劳动过程与价值形成过程的同一。
  2. 生产物价值的构成部分：
    1. 生产手段的价值：有两个条件，实际参加生产和不超过社会必要劳动时间。
    2. 劳动者的劳动时间：由抽象劳动的同一性，只以社会必要劳动时间的量来衡量。此时原料可以被视为一定量劳动的吸收器。
  3. 劳动力的价值和它在劳动过程中的价值增殖相区分。以日来衡量，劳动力的价值是生产一日劳动力所必要的生活资料，譬如说半日的生产，而劳动价值增殖是劳动者全日工作在生产物中注入的价值，后者通常多于前者。两者价值之差就是剩余价值的来源。
  4. 简单来说，价值增殖过程是延长到一定点或者说必要时间以外的价值形成过程。
  5. 资本主义的生产过程是劳动过程与价值增殖过程的统一。
  6. 复杂高级的劳动包含更大的教育费用，生产这种劳动力曾消费更多的劳动时间，因此有更高的价值。
